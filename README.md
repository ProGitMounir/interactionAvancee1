## Instructions pour exécuter l'application

    *Démarrer le backend* 
        Naviguez dans le répertoire backend/ : cd backend
        puis : ./mvnw spring-boot:run

    *Démarrer le frontend*
        Accédez au répertoire appFrontEnd/ : cd frontend
        suivis de : npm i
        puis : ng serve
